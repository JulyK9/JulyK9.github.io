# const, let, var 의 차이와 쓰임

variable (변수): 값을 저장하거나 유지하는 역할

- const (상수) : 바뀌지 않는 값, 계속 유지됨, 한번 설정하면 업데이트 불가
- let : const와 비슷해보이지만, cascade 방식에 따라 업데이트하면 변경됨
- var : const 와 let 구분 없이 예전에 썼던 방식


[정리]
 1. 기본적으로 const를 쓴다.
 2. 필요할 때만 let을 쓴다.
 3. var는 쓰지 않는다.
